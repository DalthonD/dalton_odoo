POS Wallet Management

POS Wallet Management module allows the seller to create wallets for the customer in POS Session. Customer can get their wallet recharge to add wallet credits at any time and these wallet credits can be used by the customer for future purchases. With the help of this module seller does not need to take payment as cash every time, he can use customer wallet credits as payment. This module is best for regular customers i.e Customer can recharge their wallet at any time in POS Session and use their wallet credits for future payment.

Features
  • Seller can create customer wallet to store customer credits and these credits can be used by the customer for future purchases
  • Seller can recharge wallet at any time in POS Session
  • Transaction history will be maintained for both credits and debits done in wallet
  • Seller can recharge wallet or use wallet balance in offline mode also
  • Module works even when the POS is offline.
  • No complex backend configuration
  • POS Session View
  • Backend Configuration
